## Installation
> Before running the project locally make sure you have Node.js version 16 or higher installed on your computer

To run the project locally:

1. Clone the repository
   ```bash
   git clone https://github.com/BraedenLyman/portfolio.git
  
2. Navigate to the the project folder
   ```bash
   cd portfolio
   
3. Inside the project folder install the project's packages
   ```bash
   npm install
   
4. Start the local server
   ```bash
   npm start

## Usage

The portfolio website is used to present my professional work and experience in a clear, recruiter-friendly format.

Employers can:

- View my featured projects with descriptions and live demos  
- Explore my technical skills and technologies used  
- Navigate through sections such as About, Projects, and Contact  
- Access external links to my Email, LinkedIn, GitHub and Facebook accounts
